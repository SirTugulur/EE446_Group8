# EE446_Group8
Final project Github for EE446 Group 8's frisbee throw classifier.

To install the model library, download the zip in the firmware folder, then in Arduino IDE go to Sketch -> Add ZIP library.

Attaching the board:
- Wire the Nano with the A23 12V battery and the switch.
- Find the center of mass of the bundle, and attach that to the frisbee with the Vin and Ground pins parallel to the frisbee (the board is sideways). Keep a piece of cardboard or foam between the frisbee and the board.
- Use packing tape and double sided tape to secure everything.

Installing the app:
- Using the app requires Flutter and the respective device IDE (Xcode/Android Studio).
- Run the code using flutter commands (flutter doctor to troubleshoot, flutter run, etc.).
- Use Xcode/Android Studio to install the app onto your phone
